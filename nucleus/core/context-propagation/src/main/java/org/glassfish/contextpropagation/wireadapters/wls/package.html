Provides public interfaces and classes for compatibility with Web Logic weblogic.workarea. The preferred programming model is to use the APIs in org.glassfish.contextpropagation.

Interfaces

  • PrimitiveContext
  • PropagationMode
  • Context
  • ContextMap

Removed Interfaces

The following interfaces are not needed nor supported in BG. They have been removed.
  • ContextInput
  • ContextOutput

Classes

  • AsciiContext
  • LongContext
  • StringContext
  • SerializableContext
  • PrimitiveContextFactory

Exceptions

  • NoContextException
  • PropertyReadOnlyException
